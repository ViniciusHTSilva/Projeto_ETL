# Projeto_ETL
Projeto rápido de ETL 
Este projeto tem como objetivo praticar um fluxo completo de ETL

🐳 Docker:  Hospedagem no PostgreSQ

🛠️ DBeaver: Consultas

🐍 Python: Extração, transformação e carregamento para PostgreSQ

Com esse projeto vamos conseguir responder a Querys no Banco de dados para Analise 
1 — Quais os 5 estados com mais cervejarias?

2 — Qual o tipo de cervejaria mais comum?

3 — Quantas cervejarias têm website vs não têm?
